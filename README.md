# AI 视频制作工作流

基于通用 AI Agent + Skill + Seedance 2.0/1.5 的完整 AI 视频制作流程，专注于将小说、故事转化为多集视频系列。

## 项目简介

本项目提供了一套完整的 AI 视频制作工作流，通过三件套工具实现从故事到成片的自动化生产：

- **任意 AI Agent（Claude/Codex/Gemini 等）** - 剧本创作与分镜脚本生成
- **Nana Banana Pro** - 角色、场景、道具素材图生成(可选其他生图ai)
- **Seedance 2.0 / 1.5** - 视频生成（2.0支持延长，1.5使用首尾帧衔接）

## 工作流程

```
构思主题 → 写剧本 → 生成素材描述 → 生图 → 写分镜脚本 → 逐集生成视频
```

### 详细步骤

1. **剧本创作** - 将源材料改编为四幕结构剧本（起承转合）
2. **素材规划** - 创建编号的角色(C)、场景(S)、道具(P)生成提示词
3. **图像生成** - 使用 Nana Banana Pro 生成统一风格的视觉素材
4. **分镜脚本** - 生成 Seedance 2.0/1.5 时间轴格式提示词
5. **视频生成** - 2.0使用视频延长；1.5使用首尾帧与转场运镜承接

## Seedance 2.0/1.5 提示词格式

每个分镜脚本包含三部分：

### 1. 素材清单

```
| 素材槽 | 文件 | 说明 |
|--------|------|------|
| 图片1 | C01 | 角色参考 |
| 图片2 | S01 | 场景参考 |
```

### 2. Seedance Prompt（时间轴格式）

```
水墨武侠风格，9:16竖屏，灰白墨黑色调

0-3秒画面：高空俯拍，场景建立
3-6秒画面：镜头推进，主体引入
6-9秒画面：情节发展，冲突展现
9-12秒画面：高潮时刻，情感爆发
12-15秒画面：结尾落版，氛围延续

【声音】配乐风格 + 音效 + 对白
【参考】@图片1 角色，@图片2 场景...
```

### 3. 尾帧描述

记录本集最后一帧画面，用于下一集衔接。

## 集数串联（2.0 与 1.5）

### Seedance 2.0（视频延长模式）

从第 2 集开始，使用视频延长功能保持连续性：

1. 上传上一集视频作为 @视频1
2. 在提示词开头添加：`将@视频1延长15s`
3. 这样可以创建各集间的无缝过渡

```
片头(新生成) → E1(新生成) → E2(延长E1) → E3(延长E2) → ...
```

### Seedance 1.5（首尾帧承接模式）

1. 1.5 不支持视频延长，只能用首帧/尾帧串联
2. 若可直接承接，下一段开头必须写：
   - `上一段视频尾帧，也就是本次视频的首帧：[...]`
   - `请承接上一段的[具体运镜]转场运镜，继续[...]`
3. 若不可直接承接：
   - 上一段最后1-2秒先写转场运镜
   - 下一段0-3秒明确承接该转场运镜，再落主动作
4. 承接时保持一致：方向、速度、景别、光线、主体朝向

## 风格一致性

所有素材生成提示词必须以统一的风格前缀开头，例如：

```
Chinese ink wash painting style mixed with anime cel-shading
```

角色辨识度通过不同的颜色方案和视觉标记来区分。

## 重要限制

- **最多 9 张图片** 每次 Seedance 2.0 生成
- **最多 3 个视频**（总时长 15 秒）作为参考
- **Seedance 1.5 无视频延长**，跨集必须依赖首尾帧与转场运镜
- **敏感词** 可能导致生成失败（如"江湖人士"等）
- **视频编辑** 能力有限，大多数修改需要重新生成
- **指令遵循** 复杂提示词（300+字）可能不一致

将 `skills/seedance-storyboard-generator/` 挂载到你使用的 Agent 技能目录后，输入故事需求即可生成脚本与分镜。

```text

2) 素材清单
3) 分集分镜
4) 输出 Seedance 1.5 首尾帧承接版提示词
```

## 常见问题

### Q: 如何处理敏感词问题？

A: Seedance 平台不会明确告知哪个词触发屏蔽。建议使用二分法排查：将提示词分段测试，逐步缩小范围。积累常见敏感词库，在生成时直接规避。

### Q: 如何保证角色一致性？

A: 为每个角色生成多张不同角度和状态的参考图，在分镜脚本中明确指定使用哪张图片作为参考。使用统一的风格前缀。

### Q: Seedance 1.5 没有视频延长，如何保证过渡？

A: 用首尾帧+承接运镜。下一段开头要写“上一段视频尾帧，也就是本次视频的首帧…请承接上一段的XX运镜继续…”。如果不能直承，就在上一段结尾先写转场运镜，下一段开头承接该转场运镜。
